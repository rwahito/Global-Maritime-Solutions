# Global Maritime Solutions — Terminal Operations Analysis

This project analyzes 15,000 cargo movements across 50 global terminals (2021–2024) 
to identify operational anomalies and inefficiencies triggered by the March 2021 
Suez Canal blockage. Using a star-schema dataset spanning four regional hubs 
(EMEA, APAC, AMER, LATAM), the analysis uncovers terminal bottlenecks, vessel 
performance gaps, and the long-tail disruption impact across global shipping routes.
